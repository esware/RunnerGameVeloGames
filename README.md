## Nasıl Başlayabilirim?

Proje üzerinde çalışmaya başlamak için aşağıdaki adımları izleyebilirsiniz:

1. Depoyu kendi bilgisayarınıza klonlayın.
2. Gereksinimler:
- Unity 2021.3.15f1 sürümü
3. Oyunu çalıştırın ve keyfini çıkarın!
